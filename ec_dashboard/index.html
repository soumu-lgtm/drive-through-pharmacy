6. 利益比較タブ

機能データソース計算式/参照
販売数量（全チャネル） unified_orders (allOrders) Shopify + 社割 + 卸売 の quantity 合計
販売数量（EC） unified_orders (allOrders) channel === 'Shopify' の quantity 合計
計算モード: シンプル（デフォルト）
利益計算 販管費シート (sgaData) 売上 - (販管費 × 数量)
※販管費列は全コスト含む
計算モード: 詳細（チェックボックスON時）
利益計算（詳細） 販管費シート + 送料一覧 売上 - (原価 × 数量) - 注文単位コスト - 送料
原価（数量倍） 販管費シート 製品代 + 容器代 + 化粧箱代 + イニシャル
注文単位コスト 販管費シート 配送箱代 + 決済手数料 + ピッキング費
※数量に関係なく固定
送料 送料一覧 (shippingData) 都道府県別（北海道:佐川 / 他:日本郵便）
共通設定
販管費キー（EC） 販管費シート 定価_商品名 行を参照
販管費キー（社割） 販管費シート 社割購入_商品名 行を参照
販管費キー（卸売） 販管費シート 定価_商品名 行を参照
前期計算 JavaScript 選択期間と同じ日数を開始日から遡って計算
日付フィルター profitStartDate / profitEndDate ダッシュボードとは独立
チャネル切替 profitEcOnly チェックボックス OFF: 全チャネル(EC) / ON: ECのみ
計算モード切替 profitDetailedMode チェックボックス OFF: シンプル計算 / ON: 詳細計算
JS関数: calculateProfitForPeriod(orders, start, end, ecOnly, detailedMode), applyProfitDateFilter(), resetProfitDateFilter()
チャネルマッピング: channelSgaKeyMap (Shopify→定価, 社割→社割購入, 卸売→定価)
Python関数: calculate_weekly_profit_comparison() (初期データ・送料データ生成)
